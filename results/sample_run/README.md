# Sample Run Outputs

These files were generated by running the pipeline on four uploaded example CSV files:

- one control baseline EEG file
- one control VR EEG file
- one triathlete baseline EEG file
- one triathlete VR EEG file

The outputs demonstrate that the code runs and creates the tables and figures needed for the Technical Validation section. The raw EEG CSV files are not included in this GitHub-ready package.
