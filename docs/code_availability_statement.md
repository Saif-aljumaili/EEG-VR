# Code Availability Statement

The analysis code used to inspect the raw EEG CSV files, generate quality-control summaries, apply 0.5–40 Hz band-pass filtering, segment continuous EEG into 1-s epochs, compute FFT-based theta-band summaries, and generate representative validation figures is available in the GitHub repository:

**https://github.com/YOUR_USERNAME/eeg-vr-dataset-code**

The repository contains MATLAB and Python implementations of the processing workflow, metadata templates, a channel map, example output tables, and documentation. Raw EEG recordings are not hosted on GitHub and should be deposited in the associated data repository, for example Zenodo/Figshare/OpenNeuro, with a persistent DOI.

Replace `YOUR_USERNAME` and add the final repository DOI before submission.
